# 桥洞涂鸦施工手册：万能青年旅店封面 (6层高精度版)

**目标尺寸：1.5m x 1.5m | 单张图纸：50cm x 50cm**

## 1. 准备工作
*   **输入图片**：请确保文件夹内有 `ori.png`（高清原图）。
*   **运行脚本**：双击 `stencil_maker.py` 生成图纸。
*   **打印要求**：脚本会将 1.5m 的大图切分为 **3行 x 3列 (共9张)** 小图。每张图纸的尺寸设定为 **50cm x 50cm**。请确保打印店或您的设备支持此尺寸输出。

## 2. 图纸拼接指南
由于图像较大，每个颜色层被切分为 9 部分。
*   **文件命名规则**：`LayerName_R行_C列.png`
    *   `R1` = 第一行 (顶部)
    *   `C1` = 第一列 (左侧)
*   **拼接示意**：
    ```
    [R1_C1] [R1_C2] [R1_C3]
    [R2_C1] [R2_C2] [R2_C3]
    [R3_C1] [R3_C2] [R3_C3]
    ```
*   **建议**：打印后，按照上述顺序在地面拼好，用胶带将背面粘连，形成一张完整的 1.5m x 1.5m 大模板，再进行镂空刻画。

请按照以下代码和配方提前调好 6 桶颜料。

| 编号 | 颜色名称 | 作用 | 对应图纸文件夹 | 颜色代码 (Hex/RGB) | 丙烯/乳胶漆调色配方 (估算) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **0** | **土黄色 (Base)** | **全墙打底** | 无 (直接刷墙) | **#9d8f4b**<br>RGB(157,143,75) | **土黄 70% + 白 20% + 熟褐/黑 10%**<br>*(这是最关键的底色，耐心调)* |

## 4. 施工顺序 (由底到顶，覆盖法)

### 第 0 步：全墙打底
*   **操作**：不需要模板。直接用滚筒蘸 **【0号 土黄色】**，把 1.5m x 1.5m 的墙面涂满。
*   **等待**：必须等彻底干透。

### 第 1 步：刷极浅灰层
*   **模板**：使用 `1_HighLightGrey`。
*   **操作**：滚涂 **【1号 极浅灰】**。

### 第 2 步：刷浅灰层
*   **模板**：使用 `2_LightGrey`。
*   **操作**：滚涂 **【2号 浅灰色】**。
*   **注意**：这一层会覆盖在 Layer 1 上，阴影范围开始缩小。

### 第 3 步：刷中灰层
*   **模板**：使用 `3_MediumGrey`。
*   **操作**：滚涂 **【3号 中灰色】**。

### 第 4 步：刷深灰层
*   **模板**：使用 `4_DarkGrey`。
*   **操作**：滚涂 **【4号 深灰色】**。

### 第 5 步：刷黑色层
*   **模板**：使用 `5_Black`。
*   **操作**：滚涂 **【5号 纯黑色】**。这是最后的一层，最黑的部分。

### 第 6 步：手工修补
*   **操作**：撕掉所有模板，用小笔修补连接点。

## 5. 常见问题
*   **怎么防止颜料渗到纸背面？**
    *   **关键**：颜料一定要**干**！滚筒蘸了颜料后，在废纸板上多滚几下，滚到几乎没有流动性为止。
    *   **喷胶**：建议使用 3M 77 喷胶固定模板背面。
*   **模板怎么对齐？**
    *   每一层模板的四角都有定位标记（或者以墙面的四个角为基准），贴下一层时，确保四角和上一层的位置重合。
