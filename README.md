# 桥洞涂鸦施工手册：万能青年旅店封面

## 1. 准备工作
*   **输入图片**：请确保文件夹内有 `ori.png`（高清原图）。
*   **运行脚本**：双击 `stencil_maker.py` 生成图纸。

## 2. 颜色配方表 (Color Palette)
请按照以下代码和配方提前调好 4 桶颜料。

| 编号 | 颜色名称 | 作用 | 对应图纸文件夹 | 颜色代码 (Hex/RGB) | 丙烯/乳胶漆调色配方 (估算) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **0** | **土黄色 (Base)** | **全墙打底** | 无 (直接刷墙) | **#9d8f4b**<br>RGB(157,143,75) | **土黄 70% + 白 20% + 熟褐/黑 10%**<br>*(这是最关键的底色，耐心调)* |
| **1** | **浅灰色 (Light)** | **第一层阴影** | `1_LightGrey` | **#a0a0a0**<br>RGB(160,160,160) | **白 70% + 黑 30%** |
| **2** | **深灰色 (Dark)** | **第二层阴影** | `2_DarkGrey` | **#505050**<br>RGB(80,80,80) | **白 30% + 黑 70%** |
| **3** | **纯黑色 (Black)** | **核心轮廓** | `3_Black` | **#000000**<br>RGB(0,0,0) | **纯黑 100%** |

## 3. 施工顺序 (由底到顶，覆盖法)

### 第 0 步：全墙打底
*   **操作**：不需要模板。直接用滚筒蘸 **【0号 土黄色】**，把 1.5m x 1.5m 的墙面涂满。
*   **等待**：必须等彻底干透（手摸上去不粘手）。

### 第 1 步：刷浅灰层
*   **模板**：使用 `1_LightGrey` 文件夹里的图纸拼出的模板。
*   **操作**：贴好模板 -> 滚涂 **【1号 浅灰色】**。
*   **注意**：这一层面积最大，是人物的基础轮廓。

### 第 2 步：刷深灰层
*   **模板**：使用 `2_DarkGrey` 文件夹里的图纸拼出的模板。
*   **操作**：贴好模板 -> 滚涂 **【2号 深灰色】**。
*   **注意**：这一层会覆盖在浅灰层上面，增加立体感。

### 第 3 步：刷黑色层
*   **模板**：使用 `3_Black` 文件夹里的图纸拼出的模板。
*   **操作**：贴好模板 -> 滚涂 **【3号 纯黑色】**。
*   **注意**：这是最关键的一层（人物五官、文字），一定要固定好，防止漏墨。

### 第 4 步：手工修补
*   **操作**：撕掉所有模板后，用小毛笔蘸对应颜色的颜料，把为了固定模板留下的“连接点（白点/断点）”涂上颜色，让线条连贯。

## 4. 常见问题
*   **怎么防止颜料渗到纸背面？**
    *   **关键**：颜料一定要**干**！滚筒蘸了颜料后，在废纸板上多滚几下，滚到几乎没有流动性为止。宁可多滚几遍，也不要一次滚太厚。
    *   **喷胶**：如果有条件，在模板背面喷一点 3M 77 喷胶，贴得更紧。
*   **模板怎么对齐？**
    *   每一层模板的四角都有定位标记（或者以墙面的四个角为基准），贴下一层时，确保四角和上一层的位置重合。
